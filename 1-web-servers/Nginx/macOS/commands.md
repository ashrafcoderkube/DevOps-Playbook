### Nginx Server Commands for macOS

- **Start Nginx:**
  ```shell
  sudo nginx
  ```
- **Stop Nginx:**
  ```shell
  sudo nginx -s stop
  ```
- **Restart Nginx:**
  ```shell
  sudo nginx -s reload
  ```
- **Test Configuration:**
  ```shell
  sudo nginx -t
  ```
- **Show Version:**
  ```shell
  nginx -v
  ```
- **Show Compile Settings:**
  ```shell
  nginx -V
  ```
