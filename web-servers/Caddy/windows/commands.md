### Caddy Server Commands for Windows

- **Start Caddy:**
  ```shell
  caddy run
  ```
- **Stop Caddy:**
  ```shell
  caddy stop
  ```
- **Restart Caddy:**
  ```shell
  caddy restart
  ```
- **Reload Configuration:**
  ```shell
  caddy reload
  ```
- **Test Configuration:**
  ```shell
  caddy validate
  ```
- **Show Version:**
  ```shell
  caddy version
  ```
