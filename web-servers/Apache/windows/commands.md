### Apache Server Commands for Windows

- **Start Apache:**
  ```shell
  httpd -k start
  ```
- **Stop Apache:**
  ```shell
  httpd -k stop
  ```
- **Restart Apache:**
  ```shell
  httpd -k restart
  ```
- **Reload Configuration:**
  ```shell
  httpd -k graceful
  ```
- **Test Configuration:**
  ```shell
  httpd -t
  ```
- **Show Version:**
  ```shell
  httpd -v
  ```
- **Show Compile Settings:**
  ```shell
  httpd -V
  ```
