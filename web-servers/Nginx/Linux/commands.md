### Nginx Server Commands for Linux

- **Start Nginx:**
  ```shell
  sudo systemctl start nginx
  ```
- **Stop Nginx:**
  ```shell
  sudo systemctl stop nginx
  ```
- **Restart Nginx:**
  ```shell
  sudo systemctl restart nginx
  ```
- **Reload Configuration:**
  ```shell
  sudo systemctl reload nginx
  ```
- **Test Configuration:**
  ```shell
  sudo nginx -t
  ```
- **Show Version:**
  ```shell
  nginx -v
  ```
- **Show Compile Settings:**
  ```shell
  nginx -V
  ```
