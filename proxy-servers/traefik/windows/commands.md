# **Traefik Commands**

### **Start Traefik**
```powershell
traefik --configFile=C:\traefik\traefik.yml
```

### **Check Traefik Logs**
```powershell
type C:\traefik\traefik.log
```

### **Test Traefik Configuration for Errors**
```powershell
traefik check-config --configFile=C:\traefik\traefik.yml
```
